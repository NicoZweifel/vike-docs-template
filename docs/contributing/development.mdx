### Contributing

There are no guidelines at the moment, just make sure to format everything.

Configuring `eslint` in your editor of choice should be enough.

### Commands

A list of useful commands

#### Install

`pnpm i`

#### Development

`pnpm dev`

#### Building

`pnpm build`

#### Preview build output

`pnpm preview`

#### Format

`pnpm format`

#### Lint

Will format linted files as well

`pnpm lint`

### Tools

Knowing the following tools will help immensely when contributing.

- [`vike`](https://vike.dev/)
- [`preact`](https://preactjs.com/)
- [`mdx-bundler`](https://github.com/kentcdodds/mdx-bundler)
- [`prettier`](https://prettier.io/)
- [`eslint`](https://eslint.org/)
- [`tailwind`](https://tailwindcss.com/)
