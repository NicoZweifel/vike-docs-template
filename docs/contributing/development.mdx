### Contributing

There aren't any guidelines at the moment, just make sure to format everything.

### Commands

A list of useful commands

#### Install

`pnpm i`

#### Development

`pnpm dev`

#### Building

`pnpm build`

#### Preview build output

`pnpm preview`

#### Format

`pnpm format`

#### Lint

Will format linted files as well

`pnpm lint`
