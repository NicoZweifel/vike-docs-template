---
description: Change how frontmatter is processed
---

Frontmatter can be manipulated by passing a `FrontmatterProcessor`, which makes changes to the `frontmatter` object, in [`options.ts`](/configuration).

```tsx
export type Options = {
  // other options
  frontmatterProcessor?: FrontmatterProcessor;
};
```

```ts
export type FrontmatterProcessor = (
  cwd: string,
  file: string,
  baseRoute: string,
  frontmatter: UnknownFrontMatter
) => void;
```
