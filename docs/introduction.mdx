A template to easily deploy content that has been authored with `MDX`.

### Features

- Uses `vike` + `preact` and `tailwind` for a tiny bundle size.
- `mdx-bundler` and `esbuild` to bundle mdx files in parallel.
- file and frontmatter based routing
- No bloat - just the basics with a focus on only adding light-weight dependencies.
- A Perfect lighthouse score.
- 100% static output, resulting in a very easy and blazingly fast deployment to GitHub pages, Cloudflare pages and many other hosting providers
