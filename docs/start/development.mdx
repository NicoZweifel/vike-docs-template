### Contributing

Contributions are welcome, there aren't any guidelines or anything, just make sure to format everything.

#### Install

`pnpm i`

#### Development

`pnpm dev`

#### Building

`pnpm build`

#### Preview build output

`pnpm preview`

#### Format

`pnpm format`

#### Lint

Will format as well

`pnpm lint`

### Tools

- `vike`
- `preact`
- `mdx-bundler`
- `prettier`
- `eslint`
- `tailwindcss`
